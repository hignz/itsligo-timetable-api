Timetables
api/timetable/{courseCode}

Returns JSON with full week class data.

Full course code with forward slashes replaced with hyphens needed.

e.g. api/timetable/SG_KSDEV_B07-F-Y2-1-(A)

Free rooms
api/freerooms/{0 or 1}

Returns available free rooms for today. Append 0 for labs or 1 for lecture rooms

e.g. api/freerooms/0

Course Codes
api/allcourses/{0 or 1}

Returns JSON with all course codes

e.g. api/freerooms/0
