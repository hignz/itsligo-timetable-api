Timetables
/api/timetable/{courseCode}/{collegeIndex}{semesterIndex}

Returns JSON with full week class data.

Full course code with forward slashes replaced with hyphens needed.

Course Code Example:
SG_KSDEV_B07/F/Y2/1/(A) => SG_KSDEV_B07-F-Y2-1-(A)
College Indexes:
  • IT Sligo: 0
  • Athlone IT: 1
  • Limerick IT: 2
Semester Indexes:
  • Semester 1: 0
  • Semester 2: 1
Example usage

e.g. /api/timetable/SG_KSDEV_B07-F-Y2-1-(A)/0/0

Free rooms
/api/freerooms/{0 or 1}

Returns available free labs or rooms for today. Append 0 for labs or 1 for lecture rooms

Example usage

/api/freerooms/0
